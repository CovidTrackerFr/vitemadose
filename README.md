Doc : https://hackmd.io/YHcjKsUzQ1-cMomOUuTpXw

## Démarrage rapide

Installation des dépendances :

```bash
scripts/install
```

Lancer le scraper (il tourne périodiquement, cf `.github/workflows/scrape.yml`):

```bash
scripts/scrape
```

Exécuter les tests :

```bash
scripts/test
```
